# Deploy a github repository to freeDNS:
Go to freedns.afraid.org and make an account.
### In the subdomains page, create any subdomain.
With the same subdomain for every entry, place each of these 4 IP's as your IP.
- 185.199.108.153
- 185.199.109.153
- 185.199.110.153
- 185.199.111.153
### In github, put the subdomain in the box under custom subdomains.
It may take an hour for it to properly deploy. Clear your browser cache for the new link every few minutes to make sure you're not viewing old 404's.
